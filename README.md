<div align="center">

一套优秀的中后台前端解决方案

</div>


## 特性

- 国际化，数据库配置国际化语言，自由切换显示语言
- 动态权限，不同权限对应不同菜单
- 优雅美观，Ant Design 设计体系
- Mock 数据，本地数据调试


## 使用

1. 下载项目代码。

```bash
git clone https://github.com/TurnerXi/react-admin.git my-project
cd my-project
```

2. 进入目录安装依赖，国内用户推荐使用 [cnpm](https://cnpmjs.org) 进行加速。

```bash
yarn install
```

或者

```bash
npm install
```

3. 启动本地服务器。

```bash
npm run dev // 同时启动前后端 端口3000
npm run dev:client // 只启动前端 端口8080
npm run dev:server // 只启动后端 端口3000
```
